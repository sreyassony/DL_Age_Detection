
****Note: In case of filepath issues in loading model/pictures, kindly download the entire folder,
          upload it in your Google Drive and the run the demo****

1. Open the Demo folder
2. Run the main.ipynb file
3. Run the cells for mounting the Google Drive and loading required packages, functions and the model.
4. Run the remaining cells for detecting age from
   i) Image - A file with name 'my_picture.jpg' must be there in the Demo folder.
   ii) Video - A file with name 'my_video.mp4' must be there in the Demo folder.
   iii) Webacm - Run the cell to open your webcam and determine age from the picture captured through the webcam.
